## content

### default

- `CTRL+UMSCHALT+\``: open terminal
- `CTRL+p`: open / search file
- `CTRL+UMSCHALT+p`: command palette
- `CTRL+W`: close tab
- `CTRL+S`: save tab

### custom

- `CTRL+UMSCHALT++`: insert snippet, e.g. code fence

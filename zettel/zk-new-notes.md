# zk new notes

Create a new note with zk. Recommende is [vim](vim.md).

## content

```bash
zk new --title "An interesting concept" [<directory>]
```

### search or create

```bash
zk edit --interactive
``` 

### use initial content

```bash
pbpaste | zk new --interactive
```

```bash
zk new --interactive < file.txt
```
